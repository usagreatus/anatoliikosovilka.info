About
-----
Sonder is a WordPress theme for personal website with minimal distraction for the user. With a responsive design using Twitter Bootstrap, Sonder theme gives a clean look for your website. For bug reporting and general theme feature requests, theme custom modifications and WordPress development contact the <a href="http://www.sinaru.com>developer</a>. For latest updates, check the developer site.

Customizable Features
---------------------
* Add header logo image.
* Add background image or color.
* Add phone number, email to the header and footer.
* Add social page links to the header.
* Three page templates
    * Homepage - To use for home page.
    * Full Width - To take the full available width of the theme. Does not show sidebar.
    * Default - Display page with default layout including sidebar.
